Вот **полный список файлов**, которые используются в твоём проекте, исходя из структуры `index.html`, логики и типичной реализации Telegram WebApp на Flask:

---

### 📁 Основные файлы проекта

```text
web/
├── bot.py                  # Главный бот + встроенный Flask-сервер
├── config.py               # Хранение TOKEN и других настроек
├── database.py             # Работа с SQLite (сохранение игроков, инвентаря и т.д.)
├── data/
│   └── players.db          # База данных игроков (создаётся автоматически)
└── webapp/
    └── index.html          # SPA-интерфейс (уже у тебя)
```

---

### 🔧 Подробное описание каждого файла

#### 1. `bot.py` — основной скрипт
- Запускает Telegram-бота через `python-telegram-bot`
- Встроенный Flask-сервер (`flask_app`)
- Обрабатывает:
  - `/start` — приветствие
  - `WebAppData` — действия из интерфейса (покупки, прокачка)
  - API-роуты: `/api/player`, `/api/create`, `/api/battle_result` и др.
- Отдаёт `index.html` по URL

#### 2. `config.py`
```python
TOKEN = "7891234567:AAHXYZ..."  # Токен от @BotFather
WEB_APP_URL = "https://fay-unrepelled-perceptibly.ngrok-free.dev"  # Твой ngrok-URL
```

#### 3. `database.py`
Управляет сохранением и загрузкой данных:
- `init_db()` — создаёт таблицу `players`
- `get_player(user_id)` — получает игрока
- `save_player(...)` — сохраняет все данные (имя, уровень, золото, инвентарь, экипировку)

Пример структуры таблицы:
```sql
CREATE TABLE players (
    user_id INTEGER PRIMARY KEY,
    name TEXT,
    level INTEGER,
    exp INTEGER,
    gold INTEGER,
    attributes TEXT,
    inventory TEXT,        -- JSON
    equipment TEXT,        -- JSON
    created_at TIMESTAMP
);
```

#### 4. `data/players.db`
- Автоматически создаётся при первом запуске
- Хранит всех игроков
- Используется через `sqlite3` в Python

#### 5. `webapp/index.html`
- Единственный HTML-файл (SPA)
- Полностью автономный: CSS, JS, логика боя, навигация
- Работает внутри Telegram WebApp
- Обменивается данными с `bot.py` через `fetch` и `tg.sendData`

### 🚀 Как запустить проект

1. Установи зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Запусти `ngrok`:
   ```bash
   ngrok http 8000
   ```
   → Скопируй URL (например, `https://fay-unrepelled-perceptibly.ngrok-free.dev`)

3. Вставь его в `config.py`:
   ```python
   WEB_APP_URL = "https://fay-unrepelled-perceptibly.ngrok-free.dev"
   ```

4. Запусти бота:
   ```bash
   python bot.py
   ```

5. Напиши боту в Telegram: `/start` → нажми **«🎮 Играть»**
